# ZapzapTicket - frontend

Projeto em desenvolvimento baseado no Whaticket porém utilizando Next.js e com dependências mais atuais para não dar conflito já que o Whaticket iniciou usando libs antigas.


## Instalação

```bash
$ npm install
```

## Rodando o app

```bash
# development
$ npm run dev

# production
$ npm run build
$ npm start
```

## Testando

```bash
$ npm run test
```
